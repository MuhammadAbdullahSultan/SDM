Thank you for using System Downtime Monitoring. Please read before use the system of Downtime and Equipment and also User.



Dashboard:
Primary page to view summary of Downtime and Equipment by Hour and Percentage. Summary of Downtime and Equipment can be exported in Excel and PDF file. There is a column below where User would select year, month, day to search. To view the Downtime description, please select any of the graph bar chart, it will pop-up the Downtime description. Refresh button will bring back to original display of graph.

 

Manage Downtime:
Displays summary of Downtime and Equipment by Hour and Percentage. In Add Downtime, select equipment followed by select type of maintenance (Corrective Maintenance, Preventive Maintenance, or Standby), start time at any period and end time. Do take note that end time would only accept current date. No future date input allowed. Once all keyed in, a new Downtime will be added.

Under Search Downtime, select equipment list, select type and select date. The graph would then plot the Downtime data that was searched.

 

Manage Equipment:
This page displays maintenance of Downtime System and Equipment. User would add new equipment to the table with correct format of Equipment Name with description that requires not more than 50 characters. User would also to insert system name and select a group based on the requirement

 

Manage User:
Create new user with Email and Password that requires more than 6 alphanumeric characters. Once created, the List of Existing Users displays the user created and would be able deactivate the account whenever required.

 

Profile:
Profile allows user to change password whenever required
