Manage Downtime

  • Manage Equipment displays maintenance of Downtime of Equipment.
  • Please add equipment first before adding downtime.
  • Click on 'Add Equipment' button to add new equipment.
  • Refer to list in FYP.xls (in Equipment tab) for list of equipment wanted to be added (optional).
  • Added equipment will automatically be listed in the table of existing equipment.
  • The list of equipment can be both deleted and edited, according to the user's need.
  • Equipment that has recorded downtime cannot be deleted. If it requires the user to delete an equipment that has downtime, delete the downtime for the specific equipment firsthand in Manage Downtime.
  • The table can be exported in Excel (.xlsx) and PDF format by clicking on the export buttons on top of the table.
  • There is an input for search filter for the table, where it filters the list of equipment according to the equipment name input by the user.
  • The table can be sort through ascending and descending order.
  • The list in the table is contained inside a pagination. The user can decide the amount of data can be viewed in one page, whether it is 5, 10, et cetera.

Manage Equipment
