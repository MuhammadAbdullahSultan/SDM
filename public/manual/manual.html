Thank you for using System Downtime Monitoring.



Dashboard: Primary navigation tool which is designed to view summary of Downtime and Equipment management and to search for downtime and equipment.

 

Manage Downtime: Please select equipment followed by type of maintenance. Select start time at any period and end time. End time would only accept current date. No future date allowed.

 

Manage Equipment: User would add new equipment to the table with correct format of Equipment Name with description that requires not more than 50 characters. User would also to insert system name and select a group based on the requirement

 

Manage User: The manage user allows to create user and the administrator would be able to activate or deactivate their account.

 

Profile: Profile allows user to change password whenever required
